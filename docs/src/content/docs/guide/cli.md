## Commands

- `walsync snapshot` - Take a one-time snapshot
- `walsync watch` - Continuously sync changes
- `walsync restore` - Restore from backup
- `walsync list` - List available snapshots

## Environment Variables

- `AWS_ACCESS_KEY_ID`
- `AWS_SECRET_ACCESS_KEY`
- `AWS_ENDPOINT_URL_S3`
