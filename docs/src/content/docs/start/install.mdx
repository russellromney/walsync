## CLI (Rust)

Install the command-line tool from Crates.io:

```bash
cargo install walsync
```

Verify installation:
```bash
walsync --version
```

## Python Package

Install from PyPI:

```bash
pip install walsync
```

Then import in your Python code:

```python
from walsync import WalSync

ws = WalSync("s3://my-bucket")
ws.snapshot("/path/to/app.db")
```

## Requirements

### CLI
- **Rust 1.70+** - Install from [rustup.rs](https://rustup.rs/)

### Python
- **Python 3.8+**
- **pip or conda**

## S3 Credentials

Walsync uses standard AWS SDK environment variables:

```bash
export AWS_ACCESS_KEY_ID=your_access_key
export AWS_SECRET_ACCESS_KEY=your_secret_key
export AWS_REGION=us-east-1
```

### For Tigris (Recommended)

```bash
export AWS_ACCESS_KEY_ID=your_tigris_key
export AWS_SECRET_ACCESS_KEY=your_tigris_secret
export AWS_ENDPOINT_URL_S3=https://fly.storage.tigris.dev
export AWS_REGION=auto
```

### For MinIO or Self-Hosted S3

```bash
export AWS_ENDPOINT_URL_S3=https://minio.example.com:9000
export AWS_ACCESS_KEY_ID=minioadmin
export AWS_SECRET_ACCESS_KEY=minioadmin
```

## Verify Setup

Test your configuration:

```bash
# CLI: List backed-up databases
walsync list -b s3://my-bucket

# Python
python -c "from walsync import WalSync; print('OK')"
```
